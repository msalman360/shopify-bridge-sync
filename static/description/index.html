Overview

Shopify Bridge Sync is a comprehensive connector that seamlessly integrates your Shopify store with Odoo ERP. Synchronize products, orders, customers, and inventory in real-time with an intuitive dashboard interface.

Installation & Setup

  1. Install the module from Apps menu
  2. Go to Sales > Shopify > Stores
  3. Click New to add your Shopify store
  4. Enter your Store Domain (e.g., yourstore.myshopify.com)
  5. Enter API Key and API Secret from Shopify
  6. Click Verify Connection
